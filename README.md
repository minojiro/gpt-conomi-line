## 使い方

```bash
# パッケージインストール
pnpm install

# サーバー立ち上げ
pnpm run server

# ngrok などでインターネッツに公開
ngrok http 3000
```

公開されたURLを LINE Developers の Messaging API 設定から、 Webhook URL に設定
https://developers.line.biz/console/channel/xxxx/messaging-api
